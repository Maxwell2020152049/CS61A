# CS61A

## 简介

本仓库存放`CS61A Spring 2021`的`Homework`、`Lab`和`Project`的实现代码。

以及存放课程相关的课件和笔者记录的笔记。

项目目录如下：

```shell
.
├── HW					# Homework的实现代码
├── Lab					# Lab的实现代码
├── Project				# Project的实现代码
└── README.md	        # 说明文档
```

## 关于`.gitignore`

```
**/*.zip
**/__pycache__
```

`git`不需要维护 **`压缩包`** 和 **`缓存文件`**
